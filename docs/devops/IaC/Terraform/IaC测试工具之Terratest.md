# IaC测试工具之Terratest



# 二 简介

Terratest 是一个开源的 Go 语言库，可更轻松地为基础设施代码编写自动化测试。它为通用基础设施的测试任务提供了各种帮助函数和模式，其中包括：

- 测试 Terraform 代码
- 测试打包器模板
- 测试 Docker 镜像
- 通过 SSH 在服务器上执行命令
- 使用 AWS API
- 发送 HTTP 请求
- 运行 shell 命令
